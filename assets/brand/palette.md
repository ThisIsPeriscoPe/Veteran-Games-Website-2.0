# Veteran Games — Brand Palette (Website 2.0)

Dark skin, gold accent. Confirmed by Joey 14 Jul 2026.

| Token | Hex | Usage |
|---|---|---|
| `--vg-black` | #0B0B0C | Page background |
| `--vg-coal` | #131315 | Section band alt background |
| `--vg-panel` | #1A1A1D | Cards, panels |
| `--vg-line` | #2A2A2E | Hairlines, card borders |
| `--vg-gold` | #BFAE7A | Primary accent, CTAs, rules, highlights (brand confirmed) |
| `--vg-gold-deep` | #9C8B5A | Gold hover / pressed |
| `--vg-white` | #FFFFFF | Headings |
| `--vg-grey` | #B9B9BE | Body copy on dark |
| `--vg-grey-dim` | #7C7C82 | Meta, captions |
| `--vg-danger` | #C0392B | Sold out / alerts only |

Slipstream skin notes
- Map `--vg-gold` to the theme's Primary color in Slipstream Theme Options (replaces the demo red).
- Map `--vg-black` / `--vg-coal` to Dark scheme background / alternate background.
- Buttons: gold fill with black text, or 1px gold outline with gold text. Uppercase labels.
